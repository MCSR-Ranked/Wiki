# Installation with [MCSR Launcher](https://mcsrlauncher.github.io/download)

## Launcher setup
After installing MCSR Launcher from the link above, open it and add your Microsoft account by going to `Accounts` âž” `Add Account`.

## Add Instance
In the launcher, go to `New Instance`, select `MCSR Ranked`, and press `Create Instance`. Finally, launch your new instance.

::: tip
"MCSR" stands for "**M**ine**c**raft **S**peed**r**unning", __not__ for "Minecraft Speedrunning Ranked".
:::
